Introduction

In this page we'll go through all the basic fundamentals of text/font styling in detail, including setting font weight, family and style, font shorthand, text alignment and other effects, and line and letter spacing.

Font Family

Font-family property allows you to specify a font (or list of fonts) for the browser to apply to the selected elements. The browser will only apply a font if it is available on the machine the website is being accessed on; if not, it will just use a browser default font.

There are different aspect to consider when applying a font family:

  • Web safe fonts: there are only a certain number of fonts that are generally available across all systems and can therefore be used without much worry. These are the so-called web safe fonts
  • Default fonts: CSS defines five generic names for fonts: serif, sans-serif, monospace, cursive, and fantasy.
  • Imported fonts from Google

It is important to supply a font stack so that the borwser has multiple fonts it can choose from, separating each font names by commas.

Another consideration is that font names that have more than one word need to be surrounded by quotes.

Font-family example:

p { color: red; font-family: Helvetica, Arial, sans-serif }
Font Size

Font size property can change the text size by applying px units, em units or rem units.

The font-size of an element is inherited from that element's parent element. This all starts with the root element of the entire document, the standard font-size of which is set to 16px across browsers. Any paragraph (or another element that doesn't have a different size set by the browser) inside the root element will have a final size of 16 px

It is best practices to use rem units to modify font size so that it only depends of the root element of the document

Font size example:

font-size: 1.5rem;
Other Font Properties

There are other properties applied to style the font of a page:

  • Font style: used to turn italic text on or off
  • Font weight: sets how bold the text is
Text Transform

It's a property that allows you to set your font to be transformed, using uppercase to make all text to capitals; lowercase to make all text lower case, and capitalize to uppercase each first letter of the words

Text transform example:

text-transform: uppercase;
Text Decoration

Sets/unsets text decorations on fonts (you'll mainly use this to unset the default underline on links when styling them). It can have values of underline, overline, line-through or none

You should note that text-decoration can accept multiple values at once if you want to add multiple decorations simultaneously

Also note that text-decoration is a shorthand property for text-decoration-line, text-decoration-style, and text-decoration-color. You can use combinations of these property values to create interesting effects.

Text decoration example:

text-decoration: line-through red wavy;
Text Shadow

You can apply drop shadows to your text using the text-shadow property. This takes up to four values: horizontal offset, vertical offset, blur radius, and the base color of the shadow.

You can also apply multiple shadows to the same text by including multiple shadow values separated by commas

Text shadow example:

text-shadow: 1px 1px 1px red, 2px 2px 1px red;
Text Align

The text-align property is used to control how text is aligned within its containing content box.

Text align example:

text-align: right;
